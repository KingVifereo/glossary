
# Static Testing

中文：静态测试。

测试工作产品，但不执行其代码。

静态测试是指无须执行被测代码，而是借助专用的软件测试工具评审软件文档或程序，度量程序静态复杂度，检查软件是否符合编程标准，借以发现编写的程序的不足之处，减少错误出现的概率。常见的静态测试方法包括文档分析与评审、设计与代码评审
、代码静态扫描等等。
