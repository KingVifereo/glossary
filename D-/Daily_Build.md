
# Daily Build

中文：每日构建。

每天（经常是晚上）对整个系统进行编译和集成的开发活动，保证随时可以使用包含所有最新变更的一致性系统。

具体来说，就是每天定时自动执行一次软件构建工作，将当前版本控制系统中的源代码检出到一个构建环境（即没有安装集成开发环境的干净机器）中，对其进行编译、链接、打包的过程。
