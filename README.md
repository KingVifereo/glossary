
# 测试术语表

软件测试术语表。

## 组织结构

内容的组织方式有两个维度，分别是按字母分类和按专题分类。

**按字母分类**

英文词条：归类时以词条首字母为准。如 QA，归类到 Q 分类下。  
中文词条：归类时以第一个汉字拼音的首字母为准。如测试，归类到 C 分类下。

**按专题分类**

在专题分类下，专题会采用链接和说明的方式来阐述一组相关的词条。
